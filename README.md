
## Getting Started

Open [Demo](https://embl-eta.vercel.app) with your browser.

 1. run the development server:
`npm  install`
if  it  shows  error  about  wrong  react  dependency  with  material ui try  running
`npm  install  --force`

 2. Run local development environment
`npm  run  start:local`
